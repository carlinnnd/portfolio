# Carlos Guerra Portfolio

Portfólio profissional em HTML, CSS e JavaScript puro, pronto para deploy na Vercel.

## Estrutura

- `index.html`
- `styles.css`
- `script.js`
- `vercel.json`
- `assets/`

## Deploy na Vercel

### Opção 1: GitHub + Vercel

1. Suba esta pasta para um repositório no GitHub.
2. Entre na Vercel.
3. Clique em **Add New Project**.
4. Importe o repositório.
5. Faça o deploy.

### Opção 2: Vercel CLI

```bash
npm i -g vercel
vercel login
vercel --prod
```

## Personalização rápida

- Textos: `index.html`
- Estilo e cores: `styles.css`
- Interações: `script.js`

## Observação

Se quiser usar domínio próprio, conecte o domínio no painel da Vercel após o deploy.
